# primer-proyecto

Primer proyecto de programación. Una calculadora en Python.

## Stack

- Python 3
- `calculadora.py` — lógica de la calculadora (operaciones matemáticas)
- `main.py` — punto de entrada ("Hola mundo desde wsl")
- `historial.txt` — registro de operaciones

## Commands

```bash
python main.py
python calculadora.py
```
